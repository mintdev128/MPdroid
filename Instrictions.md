# Instructions
## Main 
  To use toolbox just write the command you want to use, so simple.
  It can perform basic functions such as : 
* date, pwd, cd, clear, ls and base64 encryptor & decryptor.
  Just like in linux.
## CONSTR tool 
  In this toolbox can use constr to play another programms, you just need to write. 

```{host}@root:{path}# constr play [programm name]```

  Available from release 1.0 .
## More
  To see more commands use

```{host}@root:{path}# toolbox```
  
  Available from release 1.1 .
